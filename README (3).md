# 🥗 Food Finder

> A Persian nutrition tool for searching, comparing, and planning meals

ابزاری برای جستجو، مقایسه و برنامه‌ریزی غذایی روزانه — ساخته شده برای کاربران فارسی‌زبان

---

## ✨ Features

**Smart Search — جستجوی هوشمند**
با وارد کردن مقدار پروتئین و کربوهیدرات مورد نظر، بهترین غذاها رو بهت نشون میده

**Persian Name Search — جستجو با اسم فارسی**
اسم غذا رو تایپ کن و پیشنهادات آنی ببین — از غلط‌نویسی و فاصله هم پشتیبانی میکنه

**Category Filter — فیلتر دسته‌بندی**
غذاها رو بر اساس دسته‌بندی فیلتر کن: گوشت، لبنیات، حبوبات، غلات، میوه، سبزیجات، آجیل و...

**Food Comparison — مقایسه غذاها**
تا ۳ ماده غذایی رو همزمان کنار هم مقایسه کن

**Meal Planner — برنامه‌ریز غذایی**
وعده‌های روزانه‌ات (صبحانه، ناهار، شام، میان‌وعده) رو برنامه‌ریزی کن و مجموع کالری و مواد مغذی رو ببین

**BMR Calculator — محاسبه‌گر BMR**
با وارد کردن سن، وزن، قد، جنسیت و سطح فعالیت، کالری، پروتئین و کربوهیدرات مورد نیاز روزانه‌ات رو محاسبه میکنه — بر اساس فرمول Mifflin-St Jeor

**Daily Progress — پیشرفت روزانه**
نمودار زیبا برای اینکه ببینی چقدر به هدف کالری و مواد مغذی‌ات نزدیک شدی

**Favorites — علاقه‌مندی‌ها**
غذاهای مورد علاقه‌ات رو ذخیره کن و با مرتب‌سازی بر اساس پروتئین یا کالری بهشون دسترسی داشته باش

**Dark / Light Theme — تم روشن و تاریک**
با یه کلیک بین تم روشن و تاریک سوئیچ کن — تنظیم انتخابی‌ات ذخیره میشه

**Offline Ready — بدون نیاز به اینترنت**
همه داده‌ها و برنامه غذایی‌ات در مرورگر ذخیره میشن و بعد از بستن و باز کردن مرورگر هم باقی میمونن

## 🚀 Usage

No installation needed. Just open `fit4.html` in your browser.

```
open fit4.html → start using
```

## 🛠️ Built With

- **Vanilla HTML / CSS / JavaScript** — no frameworks, no dependencies
- **localStorage** — for persisting user data across sessions
- **Vazirmatn Font** — for clean Persian typography
